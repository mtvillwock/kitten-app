# kitten-app
Create and destroy Kittens. Live coded for WDI at General Assembly.

## Use
Run these commands in your Terminal.
1. ```git clone <url>```
2. ```cd <directory name>```
3. ```npm install```
4. ```nodemon server.js```
5. navigate to ```http://localhost:3000/kittens```
6. Create and delete kittens to your heart's content.
